Task Description:

In our hospital, there are more and more patients. The receptionist used to write down the visits on sticky notes. Unfortunately, the sticky notes get lost or taken away by doctors. The hospital director thinks that at the beginning a simple program for the receptionist will be sufficient.

Objectives

In this stage, your program should:

  • Show user a menu.
  • Create a new appointment.

    • Ask for the doctor's name.
    • Ask for the patient's name.
    • Ask for a date.

  • Print all appointments.
  • Delete appointment.

We will use database to store information and retrieve information. The following tables “Doctors”, “Patients” and “Appointments” should be created using JDBC (Java Database Connectivity) which is an API for connecting and executing queries on a database and SQL queries.

  • Use DriverManager class  to establish a database connection with the server via its getConnection() method.
  • Connection interface represents an established database connection (session) from which we can create statements to execute queries and retrieve results, get metadata about the database, close connection, etc.
  • Use Statement and PreparedStatement   to execute static SQL query and parameterized SQL query, respectively. 

“Doctors” table should have relationships one to->many “Appointment” table; “Patient” table should have relationships one to many “Appointment” table. Because one doctor can have many appointments as well as one patient can have many appointments to visit doctors.

So the algorithm of the stage1:

  • 1) Create a database
  • 2) Create tables

To create a new appointment:

  • 1) Check if the doctor name exists in the "doctors" table. If it exists use its id number to create an appointment. If not exist Insert doctor name in to “doctor” table and use the id number of the doctor to create anappointment. 
  • 2) Check if the patient name exists in the "patients" table. If it exists use its id number to create an appointment. If not exist Insert patient name in to “patient ” table and use the id number of the patient to create an appointment;
  • 3) Insert new  appointment  in to “appointment” table

Example of SQL query to create “Patients” table:

"CREATE TABLE IF NOT EXISTS patients( idP BIGINT AUTO_INCREMENT PRIMARY KEY, patientName VARCHAR(255) NOT NULL )";

  • To show appointments use SELECT SQL query.
  • To delete appointment use DELETE SQL query.
